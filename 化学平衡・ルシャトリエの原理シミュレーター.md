# 化学平衡・ルシャトリエの原理 シミュレーター AI 指示書

## 0. 共通事項

本仕様書は、既存の参考プロジェクト群と同じ設計思想・運用前提で動作する Web 教材の作成指示書である。個別機能に入る前に、全テーマ共通の前提を以下にまとめる。以降の章は、この共通事項を前提として本テーマ固有の要件のみを記述する。

### 0.1 参考プロジェクト

- [GEO-LEARNER](https://takayuki-minagawa.github.io/GEO-LEARNER/) — 観測視点切替 + 学習モジュールの三層構成
- [weather-front-sim](https://takayuki-minagawa.github.io/weather-front-sim/) — 時系列操作 + 観測値グラフ + 学習パネル + クイズ
- [web-planetarium](https://takayuki-minagawa.github.io/web-planetarium/) — 地点・時刻に応じた星空描画と i18n / テーマ切替

これらと同じ設計思想（操作パネル＋可視化ビュー＋学習パネル＋クイズ、i18n、ライト／ダーク切替、静的配信）を踏襲する。

### 0.2 共通仕様

- 静的フロントエンド構成とする（React + TypeScript + Vite を基本、軽量なら Vanilla JS + HTML + CSS も可）
- GitHub Actions により GitHub Pages へデプロイする
- 日本語 UI を基本とし、英語切替（i18n）に対応する
- ライト／ダークテーマ切替に対応する
- PC・タブレット・スマホ対応のレスポンシブ UI にする
- 画面は「操作パネル」「可視化ビュー」「学習パネル（解説＋クイズ）」の三層構成を基本とする
- 教育用途のコンテンツであり、実測・予報・診断など厳密な数値提供を目的にしない
- 公開リポジトリ前提の表現で README と仕様書を書き、個人情報・認証情報・秘密鍵は含めない

### 0.3 共通で再利用したいコンポーネント

- タイムスライダー、再生・一時停止・速度変更コントロール
- 学習パネル・クイズフレーム（状態に応じた解説文生成と正誤判定）
- i18n ロード機構、ライト／ダークテーマ切替
- 3D シーン基盤（Three.js + React Three Fiber）: 3D 系テーマで共用
- 2D グラフ・波形描画基盤（Canvas / D3.js）: 時系列グラフ系テーマで共用
- SVG 編集系 UI（図形選択・ドラッグ・履歴）: 図形操作系テーマで共用

### 0.4 学習パネルの共通方針

- 現在の状態・入力に対する解説を自動生成する
- 観察の着目点や誘導テキストを出す
- クイズモードを併設し、主要な誤解ポイントを問題化する
- 言語切替に追従して文言が再生成される構造にする

## 1. 目的

可逆反応の容器内で、濃度・圧力・温度を操作したときに平衡がどちらに移動するかを、分子レベルのアニメーションと濃度−時間グラフの両面で理解できる Web 教材を作成する。

高校化学の「化学平衡」「ルシャトリエの原理」は定性的な理解にとどまりがちであるため、濃度変化を連続的に目で追える体験型コンテンツとする。

- 可逆反応 A + B ⇌ C + D
- 平衡状態では正反応・逆反応の速度が釣り合う
- 濃度・圧力・温度を変えると平衡が移動する
- 平衡定数 K は温度だけで変わる
- 代表例: N₂ + 3H₂ ⇌ 2NH₃（ハーバー・ボッシュ）、2NO₂ ⇌ N₂O₄、H₂ + I₂ ⇌ 2HI

- **操作パネル**: 反応の選択、初期濃度、温度、圧力（体積）、触媒 など
- **学習パネル**: 解説＋クイズ

## 3. 必須機能

### 3.1 反応の選択

- プリセット反応
  - H₂ + I₂ ⇌ 2HI
  - N₂ + 3H₂ ⇌ 2NH₃
  - 2NO₂ ⇌ N₂O₄
  - CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O
- 係数、正反応と逆反応の速度定数 kf, kr、熱的性質（発熱／吸熱）を定義

### 3.2 容器内分子アニメーション

- 各化学種を色分けした丸で表示
- 分子がランダムに運動し、衝突で一定確率で反応を起こす
- 平衡に近づくにつれて反応数が釣り合う様子が見える
- 粒子数は数十〜数百でスマホでも動く軽量設計

### 3.3 濃度−時間グラフ

- x 軸: 時間、y 軸: 各化学種の濃度
- 平衡到達後の値を破線で表示
- 操作イベントがあった時刻に縦線を入れる（「温度上昇」など）
- 平衡定数 K の現在値と目標値を表示

### 3.4 操作による平衡移動

- 濃度追加: 任意の化学種を注入
- 濃度除去: 生成物を取り除く
- 体積変化: ピストンで圧縮／膨張（圧力変化）
- 温度変化: 加熱／冷却（発熱・吸熱で K が変わる）
- 触媒の添加: 平衡到達までの時間のみを変える
- 各操作でルシャトリエの予測と実際の挙動の対応を学習パネルに表示

### 3.5 学習パネル

- 現在の状態解説（ルシャトリエ的にどちら向きに移動するか）
- 平衡定数の計算手順表示
- クイズ
  - 操作に対する移動方向を当てる
  - 発熱／吸熱の判定
  - K に影響するもの／しないものの判定

## 5. データ・計算設計

### 5.1 速度モデル

- 濃度ベース: d[A]/dt = −kf [A]^a [B]^b + kr [C]^c [D]^d
- 分子アニメーションは定性演出用であり、実際のグラフは速度式で計算
- 平衡定数 K = kf / kr、温度で kf, kr を変える（アーレニウス簡易）

### 5.2 温度と K

- 発熱反応: 温度上昇で K 減少
- 吸熱反応: 温度上昇で K 増加
- ファントホッフ式の簡易版を使用

### 5.3 反応データ

`reactions.ts` に化学式、係数、発熱量、初期値、K の温度依存係数を定義。

## 6. アニメーションとビジュアル

- 容器内の分子運動は 30〜60 fps の柔らかい動き
- 反応が起きた瞬間に短いフラッシュ
- 圧縮時はピストンが動き、容器体積がリアルタイムに変わる
- グラフはライブ更新、グリッドと軸は薄めに
